# Contributing to Rex

Thank you for helping build Rex.

## Our Philosophy

Every contribution should make Rex:

- Simpler
- Faster
- Safer

## Before adding a feature

Ask yourself:

- Does it improve readability?
- Does it solve a real problem?
- Is it consistent with the language?

If the answer is "No", it probably doesn't belong in Rex.

## Coding Style

- Keep code readable.
- Write clear comments.
- Add tests for new features.
- Keep commits focused.
